How to add HTML-CSS-Javascript Codes to your Blog ?

This is very simple and easy you only follow below steps and add this widget in your blog.



Step 1 Log-in to your Blogger account.



Select Blog That You Want To add code



Step 2 Backup your Template.




Step 3 Click on The Template







Step 4 Click on The >> Edit HTML  >> Proceed.

</div>
</div>

But, It is not easy to find a code manually.So Use Ctrl + F to find.
See The Image Below







1. Hold Ctrl on the keyboard and press F,(Ctrl+F is the shortcut to reach Find Bar in almost all browsers)
2. A Find Bar will display, Type the code you want to search.
3. At once that code will highlight on the Page/Blog
